Hourly and daily data for UPC stations P1, P5, PB, PC, PJ, PK, PL and Geonor 2019-2025.

StationReview.pdf is a report on each station evaluating measurements and describing 
the procedures used to create the files in the folders

Folders 2019 to 2025:
Contain data specific to each year for all staions operating in that year.
Data are in Daily and Hourly spreadsheets and include the original and edited data. 
The spreadsheets have a Meta worksheet describing the measurments and issues in that year.
The daily files include a worksheet with graphs of the variables.
The daily and hourly files include columns with edited values as necessary and NAN indciating missing data.

Sumry folder:
Summary of selected daily data for a station from 2019 to 2015 with a worksheet to graphically
view a 7 year time series of each variable. Stations PL and Geonor are combined.

csv folder in Sumry folder:
csv files of stations P1, P5, PB, PC, PJ, PK, PL to graphically compare individual variables
for all stations using the R-code UPC_Timeseries.R
